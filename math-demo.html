Fundamental Theorem of Arithmetic: Every integer greater than 1 either is a prime number itself or can be represented as a unique product of prime numbers.
(1) $$n = p_1^{a_1} \cdot p_2^{a_2} \cdot \ldots \cdot p_k^{a_k}$$
where $p_1, p_2, \ldots, p_k$ are distinct primes and $a_1, a_2, \ldots, a_k$ are positive integers.
Big O Notation: A function $f(n)$ is said to be $O(g(n))$ if there exist positive constants $c$ and $n_0$ such that:
(2) $$f(n) \leq c \cdot g(n) \text{ for all } n \geq n_0$$
Euclid's Lemma: If a prime $p$ divides the product $ab$, then $p$ divides $a$ or $p$ divides $b$.
Let $p$ be a prime number and suppose $p \mid ab$.
If $p \mid a$, then we are done.
If $p \nmid a$, then since $p$ is prime, we have $\gcd(p, a) = 1$.
By Bézout's identity, there exist integers $x$ and $y$ such that $px + ay = 1$.
Multiplying both sides by $b$: $pxb + aby = b$.
Since $p \mid ab$, we have $p \mid (pxb + aby)$, therefore $p \mid b$.
Euclidean Algorithm

function gcd(a, b):
    while b ≠ 0:
        temp = b
        b = a mod b
        a = temp
    return a

// Time complexity: O(log min(a, b))
Complexity Analysis

The time complexity of the Euclidean algorithm is O(log min(a, b)), making it highly efficient for computing greatest common divisors.

Space complexity: O(1) for the iterative version.
